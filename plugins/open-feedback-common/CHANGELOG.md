# Changelog

All notable changes to this project will be documented in this file.

## 0.4.3

### Minor Changes

Added `openFeedbackNavPermission` (`open.feedback.nav`, action: `read`) to control visibility of the open-feedback nav item in the sidebar. Deny this permission to hide the plugin from the sidebar for a given user or group. The permission is included in the `openFeedbackPermissions` array.
